Doc Ref: [https://www.ibm.com/docs/en/cloud-paks/cloud-pak-aiops/4.11.0?topic=policies-assign-runbook-alert ](https://www.ibm.com/docs/en/cloud-paks/cloud-pak-aiops/4.11.0?topic=policies-assign-runbook-alerts)
